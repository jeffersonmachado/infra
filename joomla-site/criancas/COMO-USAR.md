# 🚀 Como Usar o Jogo "Toque na Cor"

## ⚡ Método Mais Rápido

1. **Clique duas vezes** no arquivo `index.html`
2. **Pronto!** O jogo abre no navegador

## 📱 Para Melhor Experiência (PWA)

1. **Abra o terminal** na pasta do jogo
2. **Execute**: `python -m http.server 8000`
3. **Acesse**: http://localhost:8000
4. **Instale como app** usando os botões de instalação

## 🎮 Como Jogar

1. **Clique em "Começar Jogo"**
2. **Escute** a cor solicitada
3. **Toque** no círculo da cor correta
4. **Celebre** os acertos!

## 🎵 Sons (Opcional)

Substitua os arquivos em `sounds/` por MP3s reais:
- `acerto.mp3` - Som de comemoração
- `tente_novamente.mp3` - Som de erro
- `festa_aniversario.mp3` - Som de festa
- `aplausos.mp3` - Som de aplausos
